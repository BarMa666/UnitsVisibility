Программа по вычислению юнитом количества других видимых юнитов.

сборка:
C++14
msvc v142 
Win SDK 10.0.19041.0

Особенности:
1)	Файл с исходными данными предполагает следующий формат:
	Имя файла input.txt 
	Sector:180	- сектор обзора, число с плавающей точкой, проверки значения не предусмотрены
	Distance:2	- дальность обзора, беззнаковое целое число 
	*Имя юнита*:(*абсцисса позиции, целое*,*ордината позиции, целое*):*абсцисса направления зрения, число с плавающей точкой*,*ордината направления зрения, число с плавающей точкой*)

	Пример:
	Sector: 90
	Distance: 3
	Unit1:(1,1):(1,0)
	Unit2:(1,2):(0,1)
	Unit3:(0,1):(0,0.5)

	Для тестирования файл с исходными данными имеет измененный формат
	Имя файла test_input.txt 
	*Имя юнита*:(*абсцисса позиции, целое*,*ордината позиции, целое*):*абсцисса направления зрения, число с плавающей точкой*,*ордината направления зрения, число с плавающей точкой*)?*число видимых юнитов*
	
	Пример:
	Sector: 90
	Distance: 3
	Unit1:(1,1):(1,0)?0
	Unit2:(1,2):(0,1)?0
	Unit3:(0,1):(0,0.5)?0

2) 	Результат работы приложения отображается в консоли и записывается в файл
	
3) 	Результат выводится в порядке расположения юнитов на поле, размер которого закодирован в программе:
	Для обычной сборки размер поля 100*100
	Для тестовой сборки размер поля 3*3
	
4) 	Расположение нескольких юнитов в одной ячейке поля не предусмотренно

5)	Сообщение при тестировании, выводится в случае несоответствии полученного числа видимых юнитов ожидаемому числу:
	Mismatch found: *имя юнита* expected: *ожидаемое число видимых юнитов* result: *итоговое число видимых юнитов*
	
6)	Тестовый режим включается определнием препроцессора _TEST
